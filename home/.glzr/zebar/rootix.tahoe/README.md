# Tahoe

A [Zebar](https://github.com/glzr-io/zebar) theme matching the macOS Tahoe style.

It is configured to work with [GlazeWM](https://github.com/glzr-io/glazewm) and supports the macOS Light and Dark Appearance.

It configures itself behind the Notch of MacBooks and behind the native Menu Bar which you might want to only be present on hover.
